المهمة 4: برنامج Java لإدخال بيانات المستخدم

الكود البرمجي:


import java.util.Scanner;

public class task42 {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        
        System.out.println("Enter your name: ");
        String x = in.nextLine();
        
        System.out.println("Enter your age: ");
        int y = in.nextInt();
        
        System.out.println("Enter your favorite number: ");
        int z = in.nextInt();
        
        System.out.print("your name is " + x + ",");
        System.out.print(" you are " + y + " years old,");
        System.out.print(" and your favorite number is " + z);
    }
}

شرح البرنامج:

  1. يستخدم الصنف Scanner لقراءة مدخلات المستخدم.
  2. يطلب من المستخدم إدخال اسمه (x)، عمره (y)، ورقمه المفضل (z).
  3. يقرأ هذه المدخلات ويخزنها في المتغيرات المناسبة.
  4. يطبع جملة واحدة تحتوي على جميع المعلومات التي أدخلها المستخدم.

ملاحظة: هذا البرنامج لا يغلق الـ Scanner في النهاية، وهو ما قد يُعتبر ممارسة جيدة في البرامج الأكبر لتجنب تسرب الموارد.
